Страница 2 - http://selenium1py.pythonanywhere.com/ru/basket/
Пользовательский сценарий 1.1 - Корректировка выбранных товаров в "корзине".
Тестовый сценарий 1.1.1 - Пользователь добавляет колличество единиц товара в "корзину".
Пользовательский сценарий 1.2 - Оформление товаров.
Тестовый сценарий 1.2.1 - Пользователь нажимает кнопку "Перейти к оформлению", находясь на странице "корзина".

Тестовый сценарий 1.1.1 - Пользователь добавляет колличество единиц товара в "корзину".
Предусловия: 
Пользователь находится на странице "http://selenium1py.pythonanywhere.com/ru/basket/". В "корзину" добавлен как минимум один товар.

Шаги:
Шаг 1, проверка - Пользователь один раз нажимает кнопку "вверх" в поле, в котором отображается текущее количество товаров.
Ожидаемый результат - Количество единиц товара увеличитось на 1.
Шаг 2, проверка - Пользователь нажимает кнопку "Обновить".
Ожидаемый результат - Товар увеличивается на одну единицу. В полях "Стоимость корзины теперь составляет", "Всего"(напротив выбранного товара) и "Всего в корзине (со скидками)" появляется увеличенная на стоимость товара сумма.

Тестовый сценарий 1.2.1 - Пользователь нажимает кнопку "Перейти к оформлению", находясь на страницы "корзина".
Предусловия: 
Пользователь находится на странице "http://selenium1py.pythonanywhere.com/ru/basket/". В "корзину" добавлен как минимум один товар.

Шаги:
- Шаг 1, проверка - Пользователь нажимает кнопку "Перейти к оформлению".
- Ожидаемый результат - Открывается страница оформления товаров (http://selenium1py.pythonanywhere.com/ru/checkout/).
